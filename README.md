# Projeto: Validação e Refinamento de Protótipo

Este projeto apresenta o processo de apresentação, teste e validação de um protótipo de interface desenvolvido no **Figma**, utilizando o **Gemini** para o suporte na criação de perguntas norteadoras de experiência do usuário (UX).

---

## 1. Apresentação do Protótipo
O objetivo deste protótipo é resolver o problema de usabilidade mapeado para a nossa persona. A interface foi apresentada a colegas de equipe para avaliar se o fluxo de navegação está claro, intuitivo e alinhado com os objetivos do projeto.

---

## 2. Perguntas Norteadoras (Apoio do Gemini)
Para guiar os testes com os usuários e coletar feedbacks úteis, estruturamos as seguintes perguntas com o auxílio da IA:

1. **Primeira Impressão:** Ao olhar para a tela inicial, o que você acha que este sistema faz?
2. **Navegação:** Se você precisasse interagir com o botão principal ou mudar o tema da página, onde você clicaria?
3. **Clareza:** Teve alguma parte do fluxo que pareceu confusa ou difícil de entender?
4. **Visual:** As cores e contrastes estão confortáveis para leitura?

---

## 3. Análise de Comentários e Feedbacks
Após realizar a apresentação com os testadores, coletamos os seguintes pontos de melhoria:

*   **Feedback 1:** "O botão de alternar tema no modo escuro estava com pouco contraste." -> **Ação:** Ajustar as cores.
*   **Feedback 2:** "A transição de telas foi muito rápida, deu um tranco visual." -> **Ação:** Adicionar transição suave (ease).
*   **Feedback 3:** "Gostei do efeito de profundidade nos cartões." -> **Ação:** Manter o design atual.

---

## 4. Ajustes e Entrega Parcial
Com base na análise acima, os seguintes refinamentos foram aplicados diretamente no protótipo final:
*   Ajuste de contraste de cores para o modo escuro, garantindo melhor acessibilidade.
*   Aplicação de propriedades de transição suave e efeitos de `box-shadow` para melhorar a usabilidade e a estética moderna da interface.

O projeto foi atualizado e está pronto para a entrega parcial!
